MO PROJECT SAU KHI GIAI NEN ZIP
================================

1) Giai nen ZIP ra Desktop (chi 1 tang folder, khong bi long folder 2 lan).
   Trong folder phai thay ngay: build.xml, nbproject, web, src, lib.

2) Cai dat:
   - JDK 17
   - NetBeans 17
   - Apache Tomcat 10.1 (khong dung Tomcat 9)
   - MySQL

3) NetBeans: File -> Open Project -> chon folder WarehouseManagementSystem(WMS)
   Ten project: WarehouseManagementSystem(WMS) (icon qua dia cau).

4) Loi do jakarta.servlet (import do):
   - Chuot phai project -> Properties -> Run
   - Server: chon Apache Tomcat 10.1 -> OK
   - Clean and Build (Shift+F11)

5) MySQL:
   - Bat MySQL (port 3306), chay sql/schema.sql
   - NetBeans: Source Packages -> config -> DBConfig.java
   - Sua URL, USERNAME, PASSWORD (giong kieu ket noi SQL Server)
   - Clean and Build roi Run

6) Run (F6)
   URL: http://localhost:8080/WarehouseManagementSystem(WMS)/login
   Dang nhap: admin / admin123

Neu van loi project.xml: thay file nbproject/project.xml bang ban trong ZIP moi.
